Beginner-level User Awareness Training

Competitors will engage in activities such as:

  • Identifying Phishing Emails

  • Safe practices on public Wi-Fi

  • Password Strength Checker

  • Safe browsing practices

Intermediate-level User Awareness Training

Competitors will engage in activities such as:

  • Password Management Challenge

  • Malware Avoidance

  • Wi-Fi Security

  • Privacy Settings Workshop

  • Data Handling

Advanced-level User Awareness Training

Competitors will engage in activities such as:

  • Incident Response Procedure

  • Wi-Fi Networks

  • Spear-Phishing Simulation

  • Case Study Analysis

  • Compliance Requirements
